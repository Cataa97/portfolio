# Catalina Castillo | Portafolio Profesional

Este es el repositorio de mi portafolio profesional, desarrollado desde cero para presentar mi experiencia como **Desarrolladora Full Stack con mención en Ciberseguridad**.

El diseño es totalmente responsivo, moderno, estilizado con componentes translúcidos (glassmorphism) y optimizado para una navegación fluida con animaciones de entrada.

## 🚀 Tecnologías Utilizadas

- **Frontend:** HTML5, JavaScript (ES6+), Vanilla CSS & TailwindCSS
- **Bundler:** Vite
- **Integraciones:** Web3Forms API (Formulario de contacto activo)
- **Despliegue:** GitHub Actions & GitHub Pages

## ✨ Características Especiales

- **Diseño Premium:** Estética moderna con modo oscuro, tipografía pulida y efectos de desenfoque de fondo.
- **Visualizador de Certificados:** Modal interactivo para ver diplomas y certificaciones en tiempo real.
- **Formulario de Contacto Funcional:** Envía mensajes directamente a mi correo electrónico personal usando Web3Forms.
- **CI/CD Integrado:** Configuración automática con GitHub Actions para desplegar los cambios al repositorio.
